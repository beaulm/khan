After you enter valid javascript in the text area on the left, your code will instantly be checked against the conditions above and indicators will show the result.

"Must contain" and "Can't contain" can be any valid builder objects, in the form of a comma separated list with no spaces.

Good: VariableDeclaration,IfStatement
Bad: VariableDeclaration, IfStatement

"Match structure" will check if the code you entered on the left is a superset of the code in the textarea. It's not perfect yet ;)
